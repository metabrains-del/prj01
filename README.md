# AI Bot Landing Page

This project is a simple landing page for an AI bot. It consists of an `index.html` file and a `style.css` file.

## File Structure

- `index.html`: The main HTML file for the landing page.
- `style.css`: The CSS file containing all the styles for the landing page.

## Setup

To run this project, simply open the `index.html` file in your web browser.
